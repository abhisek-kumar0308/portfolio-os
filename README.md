## ✨ What is Portfolio OS?

Portfolio OS is an open-source collection of professional Lovable skills designed to help anyone build exceptional portfolios with AI.

Instead of starting from scratch, you can combine specialized skills that improve design quality, portfolio architecture, storytelling, and recruiter impact.

Whether you're a designer, developer, marketer, student, freelancer, or job seeker, Portfolio OS provides reusable AI skills that guide Lovable toward producing higher-quality results.

## 🌟 Why Portfolio OS?

Building a great portfolio is more than creating beautiful pages—it's about presenting your work in a way that earns trust, communicates value, and creates opportunities.

Portfolio OS combines specialized Lovable AI skills into a modular toolkit that helps you build professional portfolios faster while following modern design, UX, accessibility, and storytelling principles.

Instead of relying on one massive prompt, Portfolio OS encourages a modular workflow where each skill has a clear responsibility and can be combined with others to produce better results.

# 🚀 Quick Start

Getting started with Portfolio OS takes only a few minutes.

### 1. Clone this repository

```bash
git clone https://github.com/abhisek-kumar0308/portfolio-os.git
```

### 2. Browse the available modules

```
skills/
├── 01-ui-ux-pro-designer/
├── 02-portfolio-os/
└── 03-portfolio-reviewer/
```

### 3. Import a skill into Lovable

1. Open **Lovable**
2. Navigate to **Workspace → Skills**
3. Create a new skill
4. Copy the contents of the desired `SKILL.md`
5. Save and enable the skill

### 4. Build your portfolio

Combine multiple modules together for the best results.

Example workflow:

```
UI/UX Pro Designer
        ↓
Portfolio OS
        ↓
Portfolio Reviewer
```

## 💡 Example Workflow

Portfolio OS is designed to be modular.

Instead of using one extremely large prompt, combine focused modules together.

```text
Project Idea
      │
      ▼
🎨 UI/UX Pro Designer
      │
      ▼
🏗️ Portfolio OS
      │
      ▼
🔍 Portfolio Reviewer
      │
      ▼
✨ Final Portfolio
```

### Example

**Prompt 1**

> Use the UI/UX Pro Designer module to create a premium landing page for my portfolio.

↓

**Prompt 2**

> Apply Portfolio OS to improve the information architecture and storytelling.

↓

**Prompt 3**

> Run Portfolio Reviewer and identify weaknesses before publishing.
